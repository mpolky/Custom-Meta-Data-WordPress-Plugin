Add custom fields data to meta section when using wp-api to return json results.

== Description ==

This has been tested specifically with use of Advanced Custom Fields Version 4.4.2 and WP REST API Version 1.2.2.

== Installation ==

1. Upload `custom-meta-data.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Why did you create this plugin =

This plugin was created to easily add custom fields to the json results that wp-api returns without any coding knowledge.
